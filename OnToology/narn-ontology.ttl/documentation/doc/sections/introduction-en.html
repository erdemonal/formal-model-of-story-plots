Introduction back to ToC

The Narn Ontology provides a formal vocabulary for describing story plots. The ontology is structured into seven modules. Each module addresses different aspects of the story, such as places and spatial hierarchy https://w3id.org/narn/location/, spatial topology using the Region Connection Calculus https://w3id.org/narn/rcc/, temporal entities and Allen's temporal algebra https://w3id.org/narn/to/, social relations https://w3id.org/narn/social/, event types and participant roles https://w3id.org/narn/event/, narrative structure https://w3id.org/narn/plot/, and actions https://w3id.org/narn/action/. The ontology is instantiated from Narn i Chîn Húrin (narn-instances.ttl).
